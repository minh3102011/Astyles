    Giả lập PSP trên PC

    Download version mới nhất và phù hợp với cấu hình máy tại đây http://buildbot.orphis.net/jpcsp/

    Min Windows 32bit or 64bit / Linux 32bit or 64bit / Mac OSX CPU: Pentium 4 and up GPU: Any graphic card supporting OpenGL 2.0 and up 1GB RAM
    Cấu hình:


    Recommended Windows Vista / Windows 7 (use Jpcsp 32bit even on 64bit OS to have sound and video support) CPU: Dual core @ 2.5 GHz GPU: Always install the latest drivers 2GB RAM or more
    Máy tính của bạn cần phải có Java Runtime Environment
    Cho win 32bit: http://www.filehippo.com/download_jr...9a9acabb67d1f/
    Cho win 64bit: http://www.filehippo.com/download_jr...79bbecd737526/
    SonicStage : http://download.cnet.com/SonicStage-..._4-196458.html
    UMDGen : http://psp.dashhacks.com/system/files/UMDGen_v4.00.rar

    Sau khi down và cài đặt tất cả các bạn chạy file start-windows-amd64 ( hoac 86) đối với win 32.
    Vào Options > Configuration > Region, chỉnh ngôn ngữ là English.
    Đến tab Video chỉnh như trong hình


    media


    Tiếp theo chạy UMDGen.exe chọn open dẫn đến file ISO của game psp chỉnh đến PSP_GAMESYSDIR. Xóa 2 file prometheus.prx và EBOOT.BIN. Đổi tên File EBOOT.OLD thành EBOOT.BIN xong xuôi rồi save file ISO lại ra cái mới (ko đè lên cái cũ nhé).
    Xong rồi quăng cái file ISO vừa save lại vào umdimages trong folder giả lập psp.
    Control các phím thì các bạn ấn F11 rồi tự chỉnh sau cho phù hợp với bạn là dc
    Tất cả rồi thì file->load umd-> chọn game ->run là xong
    Test thử với Zenonia, máy mình cùi bắp card on nên speed hơi thấp tí
