      Topic này mình lập ra đễ hướng dẫn bạn nào chưa biết giả lập, bạn nào biết rồi nhưng gặp lỗi gì mình sẽ giúp đỡ hay chia sẻ kinh nghiệm thì cứ post lên nhé. Topic update thường xuyên. Mong các bạn có thời gian rảnh thì ghé thăm, quan tâm, và nếu comment thì rất cám ơn đấy . Cảm ơn vì đã đọc.
      Vào vấn đề chính nhé:

      [BOX=#FFEFD5|#8B4513|4|0|0]
      Thông tin sơ bộ về giả lập: JPCSP
      [/BOX]

      Trang chủ:https://jpcsp.org/
      Download (phiên bản được update thường xuyên):https://buildbot.orphis.net/jpcsp/
      Muốn chơi được giả lập các bạn phải có java nhé:https://jpcsp.org/index.php?p=Downloads or https://www.java.com/en/download/index.jsp or
      x86 ( 32bit ): https://download.oracle.com/otn-pub/j...ndows-i586.exe
      x64 ( 64bit ): https://download.oracle.com/otn-pub/j...indows-x64.exe
      Giả lập xài được trên 32 bit và 64 bit.
      Các phần mềm hỗ trợ:
      SonicStage.
      Đối với Jpcsp để decode âm thanh, bạn sẽ cần phải tải về và cài đặt tool sau vào thư mục JPCSP. Có khi nó lại tăng được FPS cho game.
      Sony SonicStage Download Here cần thiết để hỗ trợ cho các codec sử dụng psp.
      Sau đó vào JPCSP (nếu máy bạn 64bit thì nên dùng 32bit vì bản 64bit bị lỗi MediaEngine 8-|) chọn Options -> Configuration -> Media -> stick cái Decode audio files with SonicStage
      JPCSP laucher:
      https://www.emunewz.net/forum/showthread.php?tid=977
      Giúp bạn cải thiện được FPS (Tớ nghĩ cũng không nhìu):mohoi:. Và cải thiện được 1 số lỗi.
      Các bước đã chuẩn bị xong. CÔng việc cuối cùng là vào chiến nào.
      Cách cài đặt:
      Giải nén JPCSP ra, sau đó bạn đưa game của mình vào (thường có đuoi .iso hay .cso) thư mục udimage trong giả lập chứa Emu.


      Load game của mình và run thôi. Còn cách config như thế nào mình sẽ chỉ sau.
      Config thi cũng tùy game tùy lúc :sad::
      [Video]:
      [x] Only GE graphhics (not recommended for homebrew)
      [x] Use vertex cache
      [X] Use shaders (experimental)
      [X] Use a Geometry Shader for 2D rendering (only relevant when using shaders)
      [X] Enable VAO (only relevant when using shaders and vertex cache)
      [x] Enable saving GE screen to Textures instead of Memory
      [X] Enable decoding of indexed textures (using CLUT) in shader (only relevant when using shaders)
      [X] Enable dynamic shader generation (only relevant when using shaders rec for AMD/ATI)
      |Media|
      [x] Use Media Engine (Works only with 32bit JPCSP)
      [x] Decode audio files with sonic stage (you must have ss installed for this to work)
      [x] Use non-native fonts from flash0 folder
      |Compiler|
      [x] Use compiler (dynarec)
      [50] Maximum method size Hoặc 100 cũng được.

      IV. Hướng dẫn khắc phục 1 số lỗi cơ bản.
      Lỗi "Promethus patch"

      1.Cách gỡ bõ prometheus patch bằng chương trình UDM Gen.
      Các bạn download UMDGen về
      Giải nén và click vào UMDGen.exe sẽ hiện cửa sổ


      Click vào open, chỉnh đưởng dẫn đến ISO của game cần gỡ patch



      Sau khi đã mở file ISO các bạn chỉnh đến PSP_GAME\SYSDIR
      Xóa 2 file đánh dấu đỏ là prometheus.prx và EBOOT.BIN. File đánh dấu đen EBOOT.OLD đổi tên thành EBOOT.BIN



      Bạn đã hoàn thành việc gỡ bỏ prometheus patch. Click vào nút save.
      Chú ý: Tôi khuyên bạn không nên save vào file cũ nhé, bạn hãy save ra 1 file mới. Còn bạn nào thích file nhỏ thì có thể chuyển thành file cso (trong game có thể bi lỗi crack, nhưng không ảnh hưởng nhìu), còn khi save ra mặc định là file iso.
      2. Dùng untouch.
      Vào google: search " download untouch + 'tên game'

      Lỗi:
      Dạo này hình như các bạn hay gặp lỗi này thì phải, tớ cũng đã có hướng dẫn 1 số cậu và đã thoát khỏi lối mòn, không biết phần này giúp ích gì cho vấn đề của các cậu không đây.
      Cách khắc phục:
      Trường hợp 1: Nếu cậu nào có xài chương trình quay phim Fraps thì hãy tắt nó đi, rồi khi vao game thì hãy mở lại nhé.
      Trường hợp 2: Config không đúng cách, thường là config JPCSP laucher.

      Các cậu để ý nhé, thường thì 2 cái dòng xss và xms size là mình chỉnh: 1024 1024, 512 512,..
      Còn cái dòng System profile:
      Thì cậu xem máy cậu bao nhiêu GB, hỗ trợ ss nào, để chỉnh cho phù hợp.
      Trong hình tại sao ram có 1GB mà chỉnh được dòng trên mà không bị lỗi trên. Là vì có 3gb ram ảo.
      Thế là xong.
      Chúc các bạn thành công, đây là 2 trường hợp kinh nghiệm mà mình đã mắc phải, còn các trường hợp khác thì mình sẽ nghiên cứu thêm.
