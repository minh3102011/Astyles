    Ultra Windows 8 Theme Tool - Easiest way to install & use theme on windows 8/8.1

    Tình hình là đang ở Windows 8.1 lâu rồi không vọc vạch nay nổi hứng trở lại nên vọc tí :3.
    Mình đã tìm hiểu và tách ra patch từ phần riêng cho ai muốn cài theme.
    Tool có 5 chức năng chính:
    • Patch để sử dụng đc theme (sử dụng UltraUXThemePatcher), phần này fixed không thể bỏ chọn, nếu bạn đã có patch rồi (dùng UltraUXThemePatcher hay UxStyle thì nó sẽ tự động bỏ qua phần này)
    • Shell32.dll - Phần này patch để chuyển thanh details về giống windows 7 (Mình thích cái thanh details này, Windows 8 không có rất là khó chịu), + Patch thêm Personalize BG và header image ở Control Panel
    • ExplorerFrame.dll - Phần này patch nền cho Windows Explorer
    • Xóa thanh ribbon của windows 8 (Cái thanh phía trên to to giống thanh công cụ trong office ấy)
    • Ẩn thanh status bar. (Thanh dẹp dẹp nằm phía dưới cùng Windows Explorer, nếu bạn nào đã patch details panel thì nên ẩn cái này đi để tránh dư thừa mà xấu nữa)
