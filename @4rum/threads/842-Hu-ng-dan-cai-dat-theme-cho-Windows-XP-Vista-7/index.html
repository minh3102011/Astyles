    Xem thêm bài viết hướng dẫn theo cách cũ tại đây

    Cách 1
    Download Universal Theme Patcher

    ( Hoat động tốt trên Win7 SP1)
    -Đối với win7 bản 32 bit giải nén và chọn x86 cho bản 32 bit (x64 cho bản 64 bit) sau đó nhấp đúp vào nó và cứ nhấp ok cho đến khi hiện ra cái bản.

    Nhấn     Patch cho cả 3 lựa chọn


    –> Khởi động lại máy -> Enjoy
    -Đối với Win 7 bản 64 bit thì chạy cả patch của 32 và 64 bit , có nghĩa là chạy patch 32 bit để patch 3 file sau đó chạy tiếp patch của 64 bit để patch tiếp 2 file nữa tất cả là 5 file

    –> Khởi động lại máy     -> Enjoy

    /////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////



    Cách 2
    Hoặc sử dụng cách này


    Ngoài ra bạn có thể sử dụng     UxStyle Core để patch, nó chạy khá tốt trên Win7 SP1 điều đặc biệt là sau khi patch bạn không cần khởi động lại máy mà vẫn chạy được theme bình thường     Download UxStyle Core


    Cách sử dụng:
    Giải nén và chọn đúng phiên bản Win mà bạn đang sử dụng 32bit hay 64bit
    - Nếu bạn sử dụng 32bit thì dùng UxStyle_Core_Jul13_x86.msi.
    - Nếu là 64bit thì dùng UxStyle_Core_Jul13_x64.msi
    Click chuột phải vào patch rồi Run as Administrator.
    Cài đặt xong thì cài theme vào và enjoy ( không cần restart máy nhá )







    Cách show Personalize BG


    Viết theo hướng dẫn của Yuki
    Tải Bộ Patch này về giải nén , vào thư mục shell32 và chọn file theo theo đúng ver win mà bạn sử dụng (32 hoặc 64 bit) sau đó dùng Window Theme Installer để patch file shell32.dll
    ----------------------------------------------------------------------------
    Mở Window Theme Installer sau đó check vào ô nút Shell32.dll chọn Browse rồi chọn tới file Shell32.dll mà bạn tải về trong Bộ Patch
    sau đó ấn Install Theme để patch file

    Lưu ý:
    Nếu bạn sử dung Window Theme Installer mà bị báo lỗi hay làm không thành công thì bạn có thể sử dụng Task Ownership để patch file Shell32.dll
    cách làm tương tự như file     Themecpl.dll hướng dẫn bên dưới đây (100% thành công)


    ////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

    Bây giờ tới file     Themecpl.dll

    Đối với Themecpl.dll: thì bạn cũng làm tương tự chọn file đúng với ver windows mình sử dụng sau đó Take ownership file themecpl.dll ở     Folder System32 (Win 32-bit) (    C:\Windows\System32) hay Syswow64 (Win 64-bit) (C:\Windows\Syswow64)
    rồi đổi tên nó thành themecpl.dll.bak, rồi copy file Themecpl.dll tải về vào Folder System32 (Win 32-bit) hay Syswow64 (Win 64-bit)
    Làm xong tất cả thì bạn restart lại máy và cài theme

    Ai chưa biết     Take ownership thì xem cái này
    Hệ điều hành Windows 7 không cho phép chỉnh sửa file hệ thống dù bạn đang sử dụng quyền Administrator thực sự (mặc định để ẩn)

    Muốn chỉnh sửa, ghi đè file hệ thống bạn phải làm 1 động tác là Take Ownership nhằm nắm quyền điều khiển file hệ thống đó.Hướng dẫn chi tiết cách nằm quyền điều khiển file hệ thống trong Windows 7:

    Download Take Ownership
    Chạy file Take Ownership Add.reg (nằm trong file đính kèm bài viết) sẽ thêm vào menu chuột phải 1 dòng là Take Onwership.

    Chạy file Take Ownership Remove.reg (nằm trong file đính kèm bài viết) file này gỡ bỏ tính năng của Take Onwership.

    Để thực hiện context menu trên thư mục bấm giữ SHIFT sau đó bấm chuột phải (shift + right-click); hoặc chọn file hệ thống, click chuột phải, chọn Take Onwership.



    Ko cần khởi động lại máy tính, bây giờ bạn có thể đổi tên, xóa, hay ghi đè những file hệ thống rồi đó

    Khuyến cáo: Muốn chỉnh sửa file nào thì hãy sử dụng Take Onwership để nắm quyền sử lý file đó. Ví dụ Nếu dùng Take Onwership dành quyền kiểm soát cả folder System32. Vì nếu Sau này bạn lỡ tay xóa hay sửa file nào đó trong folder System32 mà ko biết, sẽ có thể ảnh hưởng đến cả hệ thống.






    Chú ý:
    Các bạn nên Backup lại Win hoặc những file hệ thống sẽ patch trước khi làm
    để cho an toàn nếu như có sự cố có thể khôi phục lại nhanh chóng

    Check chọn vào ô ExplorerFrame.dll >>> chọn đường dẫn tới file ExplorerFrame.dll trong thư mục mà bạn tải về
    (nhớ chọn đúng với phiên bản mà máy bạn đang sử dụng là 32 bit hoặc 64 bit)




    Sau đó ấn Install theme >>> chọn theme và tự sướng với thành quả .




    ///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


    Cách 2

    Đối với những trường hợp bạn không thể chạy , báo lỗi hay patch không thành công khi xài Windows Theme installer

    Đầu tiên bạn tải [Astyles.Org] ExplorerFrame-32-64 và Download Take Ownership
    (hướng dẫn sử dụng Take Ownership bạn xem ở phần trên Cách show Personalize BG)

    Sau đó bạn vào
    C:\Windows\System32 (đối với Win 32 bit) (với win 64-bit sẽ là thư mục sysWOW64) sử dụng Take Ownership file ExplorerFrame.dll rồi đổi tên thành ExplorerFrame.dll.bk





    làm xong bước trên bạn chép file ExplorerFrame.dll trong [Astyles.Org] ExplorerFrame-32-64 mà bạn tải về vào
    C:\Windows\System32 (đối với Win 32 bit) (với win 64-bit sẽ là thư mục sysWOW64) và restart lại máy + cài theme vào là xong
