    Bản ghost Windows 8 Developer Preview (No softs)

    Đây là bản ghost Windows 8 đa cấu hình (all main) đầu tiên của Việt Nam. Bạn sẽ hỏi bản Windows (RTM) final sẽ ra khi nào:
    Phiên bản bạn đang dùng là M3 là bản pre-beta. Bản beta sẽ ra đời cuối tháng 9 (hôm nay vẫn chưa thấy) Release Candidate và RTM theo nhiều nguồn tin sẽ phát hành vào tháng 4 năm 2012).






    [img]https://tamnhin.net/Uploaded/thanhlongtn/Images/th%C3%A1ng%209/14-9/cai%20dat.jpg/img]


    Một vài thông tin:
    - Hệ điều hành Windows 8 Developer Preview Build 8102
    - No soft, có driver
    - File Ghost nặng 2,77 GB, bung và hoàn tất vào được Win chỉ mất 10 phút.
    - Ram sau khi cài chiếm tối thiểu 400 MB (chưa đúng như Microsoft quảng cáo)
    - Chạy mượt hơn Windows 7 rõ ràng
    - Ghost xong giữa năm 2012 có bản chính thức thì mình cũng làm bản ghost mới luôn.


    Nhập key sau cho Windows 8 Developer Preview: 6RH4V-HNTWC-JQKG8-RFR3R-36498
    Thực ra key này có hiệu lực 1 năm, key đã tích hợp vào bản ghost, khi đăng nhập bạn thích nhập key hay không là tùy bạn

    Link gồm 3 loại: link 4share.vn do bạn hoangngocminh92 cung cấp, link mediafire và link share.vnn.vn (không thể nhiều link hơn được nữa)

    Code:
    https://www.mediafire.com/?thabz5hzzelto
Pass mở file ghost : songngoc
    Một số lỗi các bạn hay gặp (lỗi do các bạn chứ ko phải lỗi bản ghost) hoặc câu hỏi:
    - Không giải nén được file ghost (hãy kiểm tra mã MD5 xem có khớp với mã MD5 mình đã post không.
    - Ghost xong không boot vào windows được (do trước khi ghost bạn chưa active ổ đĩa, hoặc MBR ổ cứng bị lỗi, xem hướng dẫn ở link bên dưới)
    - Không download được 1 link nào đó (do bạn download file đó nhiều lần hoặc nhiều người download quá nên bạn bị chặn lại, vài phút sau download lại sẽ được)
    - Bạn là người mới, chưa biết bung ghost, chưa biết tạo phân vùng đĩa, chưa có đĩa boot hoặc usb boot để ghost (hãy xem link hướng dẫn dưới chữ ký mình sẽ giúp bạn)
