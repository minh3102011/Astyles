      Hướng dẫn chi tiết WSB - 7 maps out - Start Menu and Taskbar

      * Tình hình là hồi trc a Katz có post 2 pài về hướng dẫn chi tiết về làm theme trong windows 7, vs phần 2 rất là chi tiết về các thành phần trong giao diện windows. Ở phần này mình chỉ tiếp nối thêm 2 phần là Start Menu và Taskbar để các bạn có trọn vẹn một 'bản đồ' các thành phần để thiết kế theme

      Đầu tiên là Taskbar nhá

      Ghi chú: Thiết lập sau mặc định ở trạng thái taskbar nằm trên, Always Combine, large icons

      - Ở một vài thành phần có ở cả 2 mục Aero và Basic, thì để thay đổi hình ảnh bạn đổi trong Aero, nhưng để chỉnh thông số, text v.v… thì bạn làm trong mục basic của nó ^^~ và những thành phần giống y nhau dù ở Aero hay Basic thì hình ảnh và thông số đều được lưu vào mục tương ứng ở basic.


      1. Taskbar & System Tray > TaskBar > Aero > TaskBar
      <1> <2> <3> <4> tương ứng các vị trí dưới – phải – trên – trái
      2. (Phần chứa những apps, khác vs 1. là phần còn lại của taskbar) Taskbar & System Tray > TaskBand > Aero > TaskBand > BackgroundTop
      (Bottom – Top – Left – Right)

      Chỉ sử dụng cho thiết lập Always Combine, ở các thiết lập khác sử dụng 1. Phân biệt giữa large icons và small icons.
      3. Taskbar & System Tray > TaskBand > Basic > TaskBand > GroupImage > 2
      Và Taskbar & System Tray > TaskBand > Basic > TaskBand > GroupImageHot > 2

      4. Taskbar & System Tray > TaskBand > Basic > TaskBand > GroupImage > 3+
      Và Taskbar & System Tray > TaskBand > Basic > TaskBand > GroupImageHot > 3+

      5. Taskbar & System Tray > TaskBand > Basic > TaskBand > TaskItemButton
      Mọi button đều được qui định ở đây nếu chọn thiết lập Always Combine, ngược lại có thêm 3 mục phía dưới


      o Libra: Taskbar & System Tray > TaskBand > Basic > TaskBand > UncombinedTaskItemLeft
      o Design: Taskbar & System Tray > TaskBand > Basic > TaskBand > UncombinedTaskItemCenter
      o Cust: Taskbar & System Tray > TaskBand > Basic > TaskBand > UncombinedTaskItemRight
      6. Taskbar & System Tray > TaskBand > Aero > Scroll Bar

      7. Taskbar & System Tray > System Tray > Aero > Flyout > Footer
      Nên chỉnh luôn ở các mục sau:
      o Taskbar & System Tray > System Tray > Basic > Tray Notify Flyout > Footer
      o Taskbar & System Tray > System Tray > Basic > System Tray Icons Panel Flyout > <7>
      8. Taskbar & System Tray > System Tray > Basic > Horizontal > Button:Open (Vị trí khác của Taskbar là Button và trong mục Vertical sau đó)

      9. Taskbar & System Tray > TaskBar > Basic > Show Desktop Button

      10. Ngày tháng: Taskbar & System Tray > System Tray > Basic > Clock > <default>

      11. Các biểu tượng trên system tray: Taskbar & System Tray > System Tray > Basic > Toolbar > Button


      1. Phần đổ bóng: Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > BackgroundShadow

      2. Phần nền: Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > Background

      3. Phần chọn: Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > ListItem

      4. Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > LeftControl

      5. Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > CenterControl

      6. Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > RightControl
      Nếu chỉ có 1 nút thì: Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > LoneControl
      7. Phần viền quanh preview: Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > Border

      8. Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > CloseCaption

      9. Taskbar & System Tray > TaskBand > Aero > TaskBandExtendedUI > Horizontal > Thumbnails & Controls > Title



      và tiếp theo là Start Menu

      Phân chia phần nền:

      Trạng thái bình thường:

      Màu đỏ: Start Menu > Panels > Aero > Top > ProgramsList

      Lưu í là phần programsList cần phải có một phần không trong suốt để chứa danh sách chương trình

      Màu cam: Start Menu > Panels > Aero > Top > PlacesList

      Màu xanh lá: Start Menu > Panels > Aero > Top > MorePrograms

      Màu xanh dương: Start Menu > Panels > Aero > Top > OpenBox

      Màu tím: Start Menu > Panels > Aero > Top > Logoff

      Danh sách tất cả chương trình: Start Menu > Panels > Aero > Top > NSCHost
      Trạng thái mở rộng thì tương tự với tên + extended

      Trạng thái tìm kiếm:

      Màu đỏ: Start Menu > Panels > Aero > Top > SearchView

      Màu xanh lá: Start Menu > Panels > Aero > Top > MoreResults

      Màu xanh dương: Start Menu > Panels > Aero > Top > OpenBoxExtended

      Màu vàng: Start Menu > Panels > Aero > Top > LogoffExtended
      Các thành phần: (Nếu có extended như Programs, Placelist… thì nhớ chỉnh luôn)


      Trong những phần dưới đây, phần nào đã thay đổi hình ảnh ở trên thì không thay đổi hình ảnh ở đây, tại đây chỉ chỉnh thông số và text

      1. Start Menu > Panels > Basic > ProgramsList

      2. Start Menu > Panels > Basic > UserPane (uncheck Userpicture để bỏ luôn hình đại diện)

      3. Start Menu > Panels > Basic > PlacesList

      4. Start Menu > Panels > Aero > PlacesListSeparator
      Vạch ngăn cách ở Programs List: Start Menu > Panels > Aero > Top > ProgramsListSeparator
      5. Start Menu > Panels > Basic > PlacesListHover

      6. Bên trái: Start Menu > Panels > Aero > LogoffSplitButtonLeft
      Bên phải: Start Menu > Panels > Aero > LogoffSplitButtonRight
      (Đổi aero thành basic để chỉnh thông số, màu chữ…)
      7. Start Menu > Panels > Basic > MoreProgramsTab

      8. Start Menu > Panels > Basic > MoreProgramsArrow và Start Menu > Panels > Basic > MoreProgramsArrowBack

      9. Sử dụng chung vs Explorer: Address, Breadcrumb & Search > Search > SearchBox > Basic > Normal > Active > <1>


      Menu ở nút power: Start Menu > Menus > Basic > Toolbar (default là seperators, còn lại là ở button)




      JumpList: ở startmenu và khi click phải taskbar
      Phần nền:
      Màu đỏ: Start Menu > Panels > Aero > Top > JumpListUser

      Màu vàng: Start Menu > Panels > Aero > Top > JumpListSystem

      (Phần viền là Windows Frame)

      Chữ đầu group (Recent, Task): Start Menu > Menus > Aero > PlaceList:ListView > GroupHeader

      Đường kẻ: Start Menu > Menus > Aero > PlaceList:ListView > GroupHeaderLine
      Button trong jumplist:
      Một button: Start Menu > Panels > Basic > JumpListItem

      Button chia 2 phần: bên trái: Start Menu > Panels > Basic > JumpListItemSplitButtonLeft và bên phải: Start Menu > Panels > Basic > JumpListItemSplitButtonRight
      Button ở Programs List:

      Một button (Màu tím): Start Menu > Panels > Basic > ListItem

      Button chia thành 2 phần: Màu xanh lá: Start Menu > Panels > Basic > JumpListListItemLeft và Màu đỏ: Start Menu > Panels > Basic > JumpListListItemSplitButtonRight
      Mũi tên: Start Menu > Panels > Basic > JumpListArrow

      2 cái Unknow đúng là chả bik làm gì ++ nhưng thôi cũng đổi cho chắc ăn
