    HƯỚNG DẪN SỬ DỤNG RAINMETER CHO WINDOWS XP
    - Cài đặt chương trình Rainmeter.
    - Giải nén các file skin
    - Chép 2 file CoreTempPlugin.dll và MailMeter.dll (2 file này ở trong bộ HUD) vào C:\Program Files\Rainmeter (tùy theo nơi cài đặt)
    - Chép 2 thư mục Black and White (2 thư mục này ở trong bộ skin HUD) vào C:\Program Files\Rainmeter\Skins (tùy theo nơi cài đặt)
    - Chạy chương trình > nhấp chuột phải lên biểu tượng thu nhỏ của chương trình trên thanh Taskbar >Config >Lựa chọn từng cái nó sẽ hiện ra desktop.
    HƯỚNG DẪN SỬ DỤNG RAINMETER CHO WINDOWS VISTA - WINDOWS 7


    Đầu tiên, bạn down Rainmeter và skin bạn muốn dùng.
    Sau đó bạn mở folder skin vừa down ra, ở đây Thursday lấy Skin Glass làm ví dụ luôn. Sau khi down file zip về, giải nén ra, được folder Glass Them All, bạn copy tất cả các folder con trong Glass Them All vào folder Skin theo đường dẫn: My Document -> Rainmeter -> Skins.
