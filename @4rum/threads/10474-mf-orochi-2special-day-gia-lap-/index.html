    Download: https://buildbot.orphis.net/jpcsp/
    Support cả 32 và 64 bit OS ( Win + Linux )
    Play được khá nhiều game, liên tục cải thiện và fix lỗi giúp chơi được nhiều game hơn. Cụ thể một số game như KH: BBS, Dissidia, God of War...
    Còn nhiều vấn đề cần guide, well, đây là lí do cái topic này ra đời: Guide + thảo luận + report game playable.
    JPCSP - Compatibility List [Commercial Games] (Part 1)
    JPCSP - Compatibility List [Commercial Games] (Part 2)
    HƯỚNG DẪN


    Minimum

    Trích:
    Windows 32bit or 64bit / Linux 32bit or 64bit / Mac OSX
    CPU: Pentium 4 and up
    GPU: Any graphic card supporting OpenGL 2.0 and up
    1GB RAM

    Recommended

    Trích:
    Windows Vista / Windows 7 (use Jpcsp 32bit even on 64bit OS to have sound and video support)
    CPU: Dual core @ 2.5 GHz
    GPU: Always install the latest drivers
    2GB RAM or more


    Người viết chỉ mới test trên Win 7 32+64, Linux chưa vọc, nên phần này chỉ hướng dẫn cho Win.

    Trước hết đảm bảo bạn đã cài JRE ( JAVa Runtime Environment ):

    x86 ( 32bit ): https://download.oracle.com/otn-pub/j...ndows-i586.exe
    x64 ( 64bit ): https://download.oracle.com/otn-pub/j...indows-x64.exe

    Cài thêm SonicStage: https://anonym.to/?https://download.cn..._4-196458.html

    Theo khuyến cáo của team dev thì kể cả máy 64 cũng nên cài bản 32 để đảm bảo supported.

    Jpcsp được thiết kế tối ưu hóa trên nền tảng dual core ( tương thích với cấu trúc của PSP dual-core ). Với máy 4 core hoặc hơn thì cũng có tác dụng đôi chút về performance, tuy không nhiều.

    Card GPU của NVIDIA fully supported và Jpcsp sử dụng được sức mạnh của GPUs.
    Đảm bảo driver GPU updated.

    Sau khi download về máy, giải nén ra, chúng ta có ( trong hình là bản 32 )
