    Re: tạo bootscreen cho win7

    Đầu tiên bạn down cái này về


    Mã:
    Boot Animations.rar
    Trong này nó gồm các file cho Win 32bit và 64bit + số bootscreen đc làm sẵn trong folder Boot Animation

    Chọn Folder 32bit hoặc 64bit(tùy theo Win bạn xài là 64bit hay 32bit). Ở đây mình xài 64bit, nên mình sẽ giải nén Folder 64bit đc như sau
    Các bước thực hiện:(tất cả nên run dưới quyền Admin)

    1/ Chạy file AddCertToRegHive(nó sẽ chạy chương trình CMD lên nhưng chỉ chớp 1 cái là hoàn tất)
    2/ Chạy Win7BootSigner


    *Mặc định nó sẽ tự chọn file winload,nên ko cần chọn, chỉ đánh dấu vào cái Enable Testing------>tiếp dó chọn {current} Windows 7.....rồi bấm Go là xog,tắt đi

    3/Tạo file làm bootscreen(nếu muốn xài boot có sẵn thì bỏ thao tác này)

    Thông số của file như sau
    (Vì đây là Bootscreen động nên sẽ có thêm thông số Frame)
    Khung hình: 15Frame/s (tức là 1s sẽ có 15 khung hình)---->bạn có thể dùng 1 số chương trình làm Flash hay kỹ xão phim để làm cái này
    Kích thước: 200x200px cho mỗi tấm
    Độ dài Animation: 7s (4s đầu chạy 1 lần, 3s sau chạy lặp)
    ******Ở đây xuất hiện 2 cách để add bootscreen
    1/ Add nguyên folder chứa tất cả các file (200x200px)
    2/ Add file bmp chứa toàn bộ file (200x200px) chạy theo chiều dọc

    ........

    Cách 1 thì chỉ việc chọn folder chứa file là đc ko làm gì thêm nên mình sẽ bỏ qua và hướng dẫn cách 2


    Trước khi đi vào cách 2 mình có lưu ý nhỏ.

    --------Như thông số ở trên. Đoạn Animation là 15FPS và dài 7s-----> chúng ta phải làm là 15*7=105 tấm hình để đc 7s
    --------Đây là các file hình mình đã chuẩn bị sẳn, đc save với định dạng Tiff(bạn có thể save ở các định dạng khác như bmp,taga,png...nhưng png là nhẹ nhất)
    Cách 2 với 1 file bmp chứa 105 hình theo chiều đọc thì ta sẽ có kích thước của file là 200x21000.(như hình dưới)(mình xài Photoshop để làm)


    Sau đó bạn nhấp dúp vào layer background và bấm OK + tắt mắt nó đi
    Sau đó bạn Zoom lên để dễ làm việc.
    Tiếp theo bạn kéo thả từng tấm vào PTS và canh chỉnh lại sau đó Enter để kết thúc tâm đầu tiênẢnh đã được thu nhỏ. Vui lòng nhấn vào đây để xem kích thước thật 1024x576.

    Tiếp tục làm tương tự cho các tấm tiếp theo để đc như sau
    Sau khi hoàn tất save lại với tên là activity với định dạng là bmp. Hoàn tất công đoạn tạo bootscreen


    Bước tiếp theo là đổi bootscreen của Win thành bootscreen bạn vừa tạo.

    -------------------------------------------------------------------
    Chạy chương trình Win7BootUpdate lên


    1/ Chọn nút select thứ 2 chọn đường dẫn tới folder chứa file activity.bmp hoặc folder chứa 105 file hình
    2/ Bỏ chọn Text
    3/ Chọn Enable testing-------> chọn {current} Windows 7.........
    4/ Go. Hoàn tất.

    Tiếp theo file repack cái file đó lại. Chạy RebuildMUI(Run As Admin)------>Bấm Y cho nó Rebuild. Hoàn tất reset lại máy xem đã có chưa


    Lưu Ý nếu boot lỗi thì có thể đưa đĩa Win vào rồi chọn Repair Startup để sửa lỗi lại(đây cũng là cách cho nó về mặc định) sau đó tiền hành làm lại từ đầu
