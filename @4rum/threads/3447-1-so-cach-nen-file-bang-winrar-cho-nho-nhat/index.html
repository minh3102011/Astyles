    1 số cách nén file bằng winrar cho nhỏ nhất

    Sau mấy lần tìm hiểu hiện mình chỉ biết vài bước nén file bằng winrar cho nhỏ lại . Bác nào còn cách khác thì chia sẻ anh em nghen

    Đây là cách của mình

    Bạn nhấn properties vào file cần nén chọn Add to archive . Điều này chắc ai cũng rõ rùi

    Phần quan trọng nằm ở đây
    Sau đó sẽ hiện 1 cửa sổ - trong thẻ general : Ở hàng compression method bạn chọn Best ( Chất lượng nén tốt nhất siêu nhỏ)

    Nếu các bạn muốn cho file nén có thể repair nếu file hỏng thì hãy tick thêm 2 hàng
    Create solid archive và Put recovery record
    Tiếp đến ở thẻ Time: Bạn tick 2 hàng High precision modification time và Store last access time ở Mục File time to store

    Mình thực hiện cách trên đã nén được cái folder L4D 6G còn 3G ( 50% đấy các bác ạ)
    Nhưng cách này chỉ nén được với các file chưa giới hạn độ nén . Nếu nén cái file hình ảnh thì nó chỉ nén dc 1 %

    Hiện mình chỉ có cách này thui nếu bác pro nào còn cách khác thì chia sẻ anh em nghen
