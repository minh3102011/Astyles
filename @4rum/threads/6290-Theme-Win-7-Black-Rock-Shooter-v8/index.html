    k được nữa thì dùng Take ownership để thay file EF nhá .


    Hệ điều hành Windows 7 không cho phép chỉnh sửa file hệ thống dù bạn đang sử dụng quyền Administrator thực sự (mặc định để ẩn)

    Muốn chỉnh sửa, ghi đè file hệ thống bạn phải làm 1 động tác là Take Ownership nhằm nắm quyền điều khiển file hệ thống đó.Hướng dẫn chi tiết cách nằm quyền điều khiển file hệ thống trong Windows 7:

    Download Take Ownership
    Chạy file Take Ownership Add.reg (nằm trong file đính kèm bài viết) sẽ thêm vào menu chuột phải 1 dòng là Take Onwership.

    Chạy file Take Ownership Remove.reg (nằm trong file đính kèm bài viết) file này gỡ bỏ tính năng của Take Onwership.

    Để thực hiện context menu trên thư mục bấm giữ SHIFT sau đó bấm chuột phải (shift + right-click); hoặc chọn file hệ thống, click chuột phải, chọn Take Onwership.




    Ko cần khởi động lại máy tính, bây giờ bạn có thể đổi tên, xóa, hay ghi đè những file hệ thống rồi đó

    Khuyến cáo: Muốn chỉnh sửa file nào thì hãy sử dụng Take Onwership để nắm quyền sử lý file đó. Ví dụ Nếu dùng Take Onwership dành quyền kiểm soát cả folder System32. Vì nếu Sau này bạn lỡ tay xóa hay sửa file nào đó trong folder System32 mà ko biết, sẽ có thể ảnh hưởng đến cả hệ thống.
