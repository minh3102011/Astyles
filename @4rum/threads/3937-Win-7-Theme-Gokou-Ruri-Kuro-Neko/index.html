    [Win 7 Theme] Gokou Ruri - Kuro Neko






    Mới fix xong, còn nóng hôi hổi nhá

    Link:

    **Hidden Content: Thanks to see the content**
    **Nội Dung Ẩn: Ấn Nút Thanks Để Hiện Nội Dung**


    Cách show BG Folder

    Trước hết lấy patch ở đây.

    Chạy Windows Theme installer,chọn ExplorerFrame,chọn đường dẫn tới file ExplorerFrame.dll tương ứng với version win của bạn (x86 or x64 -- PLEASE REMEMBER IT),ấn Install theme --> reboot and...Enjoy! :deptrai:


    Cách show Personalize BG

    Chắc ai cũng biết xài cái Window Theme Installer rồi đúng ko. (ko có thì google cho nhanh, hoặc lấy ở trên)
    Down bộ patch đặc biệt này về.

    Special Patch

    Đối với Themecpl.dll: Take ownership file themecpl.dll ở Folder System32 (Win 32-bit) hay Syswow64 (Win 64-bit) rồi đổi tên nó thành themecpl.dll.bak, sau đó copy paste file tương ứng version win vào folder.
