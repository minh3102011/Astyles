    Habbo Hotel

    Habbo là một thể loại game mạng xã hội nhằm vào thanh thiếu niên. Game này được sở hữu và hoạt động của Tổng công ty Sulake. Game được bắt đầu vào năm 2000 và được sử dụng tại hơn 150 quốc gia tính đến tháng 8 năm 2012. Game được phát triển đầu tiên dành cho người Thổ Nhĩ Kỳ và tính đến tháng 8 năm 2011 có hơn 230 triệu thành viên đã được đăng ký có trung bình 10 triệu du khách online hàng tháng

    - Game do người dùng tùy biến
    + Tự mua cỏ về trồng rồi xây 1 sân đá bóng cho người chơi đến đá bóng
    + Tự mua tuyết về xây dựng 1 sân trượt băng cho mọi người trượt
    + Tự mua loa + đèn và sơn tường vân vân để xây dựng 1 sàn nhảy cho mọi người vào
    + Và mọi thứ trong game đều do người dùng xây dựng lên

    - Game có nhiều ngành nghề
    + Công an
    + Bác Sỹ
    + Ca Sĩ
    + VIP
    + Vân vân

    - Hình ảnh trong game

    Bộ webgame gồm 3 phần chính

    Phần 1 : Client
    Bao gồm :
    PHP Code:
    - app // Bao gồm các file về cấu hình cho client + giao diện website
    - quanly // Hệ thống admin
    - global.php
    - index.php 
    Phần 2 : Server
    Bao gồm :
    PHP Code:
    - start // Trong này bao gồm những file để chạy server móc nối người dùng với game 
    Phần 3 : Flash
    Bao gồm :
    PHP Code:
    - gordon // Trong này chứa toàn bộ dữ liệu Flash của game 
    Hướng dẫn cài đặt :
    Bước 1 :
    Giải nén toàn bộ folder ở thư mục Htdocs vừa download về bỏ vào Htdocs trên máy của các bạn
    Sau đó Restore Database del_revcms.sql nằm trong thư mục C:\xampp\htdocs\start với tên bất kỳ

    Bước 2 :
    Vào C:\xampp\htdocs\app\management\config.php chỉnh sửa cấu hình cho client

    PHP Code:
    <meta http-equiv="content-type" content="text/html; charset=utf-8" /> // dòng này để toàn bộ web hiển thị tiếng việt
    <?php
    if(!defined('IN_INDEX')) { die('Sorry, you cannot access this file.'); }

    $_CONFIG['mysql']['connection_type'] = 'pconnect'; // Kiểu kết nối

    $_CONFIG['mysql']['hostname'] = 'localhost'; // Máy chủ MySql

    $_CONFIG['mysql']['username'] = 'root'; // MySQL username

    $_CONFIG['mysql']['password'] = ''; //MySQL password

    $_CONFIG['mysql']['database'] = ''; // Tên Database mà bạn vừa restore

    $_CONFIG['mysql']['port'] = '3306'; //MySQL's port

    $_CONFIG['hotel']['server_ip'] = '127.0.0.1'; // IP của máy chủ (nếu online thì sửa thành IP của máy chủ đó)

    $_CONFIG['hotel']['url'] = 'https://127.0.0.1/'; // Link của trang chủ (lưu ý : phải có "/")

    $_CONFIG['hotel']['name'] = 'Habbo '; // Tên của Game

    $_CONFIG['hotel']['desc'] = 'Game mạng xã hội thời gian thực'; // Câu trích dẫn đi kèm

    $_CONFIG['hotel']['email'] = '[email protected]'; // Email quản trị

    $_CONFIG['hotel']['in_maint'] = false; // để false để mở website, true để bảo trì website

    $_CONFIG['hotel']['motto'] = 'Toi yeu ' . $_CONFIG['hotel']['name']; // Mặc định status của thành viên

    $_CONFIG['hotel']['credits'] = 5000; // Mặc định số tiền có sẵn khi vào game

    $_CONFIG['hotel']['pixels'] = 5000; // Mặc định số điểm có sẵn khi vào game

    $_CONFIG['hotel']['figure'] = 'hr-100.hd-190-7.ch-210-66.lg-270-82.sh-290-80'; 

    $_CONFIG['hotel']['web_build'] = '63_1dc60c6d6ea6e089c6893ab4e0541ee0/1103'; 

    $_CONFIG['hotel']['external_vars'] = 'https://127.0.0.1/gordon/external_variables.txt'; // cái này là đường link data cho game

    $_CONFIG['hotel']['external_texts'] = 'https://127.0.0.1/gordon/external_flash_texts.txt'; // File này là file việt hóa nè, các bạn chỉnh cho đúng đường dẫn theo server của mình

    $_CONFIG['hotel']['product_data'] = 'https://127.0.0.1/gordon/productdata.txt'; 

    $_CONFIG['hotel']['furni_data'] = 'https://127.0.0.1/gordon/furnidata.txt'; 

    $_CONFIG['hotel']['swf_folder'] = 'https://127.0.0.1/gordon/swf/'; // Cái này là thư mục Flash để chạy game (phải có "/" đằng sau)

    $_CONFIG['template']['style'] = 'obstinate'; // Tên của giao diện ở trong tpl/skins/

    $_CONFIG['thehabbos']['username'] = 'Kryptos';
    $_CONFIG['retro_top']['user'] = 'Kryptos'; 

    $_CONFIG['recaptcha']['priv_key'] = '6LcZ58USAAAAABSV5px9XZlzvIPaBOGA6rQP2G43';
    $_CONFIG['recaptcha']['pub_key'] = '6LcZ58USAAAAAAQ6kquItHl4JuTBWs-5cSKzh6DD';

    $_CONFIG['social']['twitter'] = 'TwitterAccount'; // Link rút gọn của Twitter
    $_CONFIG['social']['facebook'] = 'FacebookAccount'; // link rút gọn của Facebook

    ?>
    Bước 3 :
    Vào C:\xampp\htdocs\start\config.conf chỉnh sửa cấu hình cho server

    PHP Code:
    ## Chỉnh thông số như ở bước 2
    db.hostname=Localhost
    db    .port=3306
    db    .username=root
    db    .password=
    db.name= 
    Bước 4 :
    Cài đặt framwork 4.0

    Sau khi hoàn tất 4 bước trên thì bạn truy cập vào C:\xampp\htdocs\start click vào Phoenix3.7.1.exe để tiến hành chạy server sau đó vào game bằng địa chỉ https://127.0.0.1

    Chú ý :
    - Thư mục Start bạn ném ở đâu cũng được, không nhất thiết để bên trong htdocs đâu
    - Sau khi chỉnh sửa bất kỳ cái gì thì bạn nên xóa cache để game load lại những cái mới nhé
    - Mỏi Tay Quá
