    Thay nền Windows Media Player 12 bằng hình cá nhân

    Phiên bản mới WMP12 Library Background Changer 2.0 nay cho phép bạn dùng hình ưa thích làm nền cho Windows Media Player 12.
    Phiên bản tiền nhiệm của WMP12 Library Background Changer chỉ hỗ trợ đổi nền cho Windows Player 12 bằng một trong năm hình có sẵn và không được bắt mắt lắm. Do vậy, đây quả là tin vui cho những vọc sĩ đang tìm cách tùy biến giao diện Windows Media Player 12 với hình ảnh cá nhân.
    Bạn tải về miễn phí WMP12 Library Background Changer 2.0 tại địa chỉ http://winreview.ru/request.php?8 (dung lượng 2,12MB, tương thích Windows 7). Sau khi tải về và giải nén, bạn chạy file wmplibbg.exe để khởi động chương trình (trước đó nhớ đóng giao diện Windows Media Player 12 nếu đang mở).


    Trên giao diện chính của WMP12 Library Background Changer 2.0 hiện ra, bạn giữ nguyên tùy chọn Default dưới trường Choose new background, bấm nút Repace và tìm chọn hình ưa thích (định dạng JPG hay PNG; nên chọn hình chất lượng cao) dùng làm nền cho Windows Media Player 12.
    Lưu ý: Năm tùy chọn Image 1, Image 2, Image 3, Image 4, Image 5 ứng với năm hình nền có sẵn từng xuất hiện trong phiên bản WMP12 Library Background Changer 1.0.
    Xong, bạn bấm nút Apply để áp dụng.
    Bây giờ, bạn đã có thể khởi động Windows Media Player 12 để chiêm ngưỡng thành quả.


    Khi không muốn dùng hình nền cho Windows Media Player 12 nữa, bạn chọn No image và bấm Apply.
